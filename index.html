FFmpeg — библиотека с открытым исходным кодом, которая используется ровером Perseverance на Марсе и позволяет сжать видео в десятки раз.

Для этого, нужно, установив FFmpeg, в папке, где лежит видео, которое вы хотите сжать, в командной строке выполнить команду ffmpeg -i input.mp4 -vcodec libx265 -crf 32 -s 720x1280 output.mp4,
где input — название исходного видео,
output — новое название,
720x1280 — новое разрешение,
-crf 32 — степень сжатия.
